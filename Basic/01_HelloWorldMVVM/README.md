# 01 - Hello World MVVM

## 學習目標
- 理解 MVVM 的基本架構
- 了解 View 和 ViewModel 的關係
- 學習基本的 DataContext 設定

## 專案結構
```
01_HelloWorldMVVM/
├── Models/
├── ViewModels/
│   └── MainViewModel.cs
├── Views/
│   └── MainWindow.xaml
│   └── MainWindow.xaml.cs
└── App.xaml
```

## 核心概念
1. **ViewModel** 作為 View 的資料來源
2. **DataContext** 綁定 ViewModel
3. 最簡單的單向資料繫結

## 執行結果
顯示一個簡單的 "Hello, MVVM World!" 訊息視窗。
