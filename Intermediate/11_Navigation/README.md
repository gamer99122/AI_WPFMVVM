# 11 - Navigation (頁面導航)

## 學習目標
- 理解 MVVM 中的頁面導航模式
- 學習使用 Frame 和 Page 導航
- 掌握 NavigationService 的實作

## 核心概念
- Frame: 承載頁面的容器
- Page: 可導航的頁面
- NavigationService: 導航服務
- ViewModel-First Navigation: ViewModel 驅動的導航

## 導航模式
1. View-First: 直接導航到 View
2. ViewModel-First: 透過 ViewModel 導航
3. Region-Based: 區域導航（Prism 模式）
